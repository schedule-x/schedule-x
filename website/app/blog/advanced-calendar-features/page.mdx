# Advanced Calendar Features: Drag & Drop, Recurring Events, and More

Schedule-X goes beyond basic calendar functionality to provide advanced features that make your calendar application truly powerful and user-friendly.

## Drag & Drop Functionality

One of the most intuitive ways to manage events is through drag and drop. Schedule-X makes this seamless:

```tsx
import { dragAndDropPlugin } from '@schedule-x/drag-and-drop'

const calendar = createCalendar({
  target: '#calendar',
  plugins: [dragAndDropPlugin()],
  events: [
    {
      id: '1',
      title: 'Team Meeting',
      start: '2024-01-15T10:00:00',
      end: '2024-01-15T11:00:00',
    }
  ]
})

// Listen for drag events
calendar.on('eventDrop', (event, newStart, newEnd) => {
  console.log('Event moved:', event.title, 'to', newStart)
  // Update your backend here
})
```

### Customizing Drag Behavior

You can customize which events are draggable and how they behave:

```tsx
const dragAndDropConfig = {
  draggable: (event) => event.title !== 'Fixed Meeting',
  dropZones: ['timeGrid', 'dayGrid'],
  snapToGrid: true
}

const calendar = createCalendar({
  target: '#calendar',
  plugins: [dragAndDropPlugin(dragAndDropConfig)],
  // ... other config
})
```

## Recurring Events

Schedule-X supports complex recurring event patterns using the RRULE standard:

```tsx
const recurringEvents = [
  {
    id: 'weekly-standup',
    title: 'Weekly Standup',
    start: '2024-01-15T09:00:00',
    end: '2024-01-15T09:30:00',
    rrule: 'FREQ=WEEKLY;BYDAY=MO,TU,WE,TH,FR'
  },
  {
    id: 'monthly-review',
    title: 'Monthly Review',
    start: '2024-01-15T14:00:00',
    end: '2024-01-15T15:00:00',
    rrule: 'FREQ=MONTHLY;BYMONTHDAY=15'
  },
  {
    id: 'quarterly-planning',
    title: 'Quarterly Planning',
    start: '2024-01-15T10:00:00',
    end: '2024-01-15T12:00:00',
    rrule: 'FREQ=YEARLY;BYMONTH=1,4,7,10;BYMONTHDAY=15'
  }
]
```

### RRULE Patterns Explained

- **FREQ=WEEKLY;BYDAY=MO,TU,WE,TH,FR**: Every weekday
- **FREQ=MONTHLY;BYMONTHDAY=15**: Every 15th of the month
- **FREQ=YEARLY;BYMONTH=1,4,7,10**: Every quarter
- **FREQ=DAILY;INTERVAL=2**: Every other day
- **FREQ=WEEKLY;COUNT=10**: Weekly for 10 occurrences

## Timezone Support

Schedule-X provides comprehensive timezone support:

```tsx
const calendar = createCalendar({
  target: '#calendar',
  timezone: 'America/New_York',
  events: [
    {
      id: 'global-meeting',
      title: 'Global Team Meeting',
      start: '2024-01-15T15:00:00Z', // UTC time
      end: '2024-01-15T16:00:00Z',
    }
  ]
})

// Change timezone dynamically
calendar.setTimezone('Europe/London')
```

## Integration with External APIs

Schedule-X can easily integrate with external calendar APIs:

```tsx
// Google Calendar integration
const fetchGoogleEvents = async () => {
  const response = await fetch('/api/google-calendar/events')
  const events = await response.json()
  
  calendar.addEvents(events.map(event => ({
    id: event.id,
    title: event.summary,
    start: event.start.dateTime,
    end: event.end.dateTime,
    description: event.description
  })))
}

// Sync with backend
calendar.on('eventChange', async (event) => {
  await fetch('/api/events/' + event.id, {
    method: 'PUT',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(event)
  })
})
```
