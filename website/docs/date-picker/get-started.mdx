# Get started

The date picker can be downloaded from npm:

```bash
npm i @schedule-x/date-picker @schedule-x/theme-default
```

## Render the date picker

```js
import { createDatePicker } from '@schedule-x/date-picker'
import '@schedule-x/theme-default/dist/date-picker.css'

const datePicker = createDatePicker(document.querySelector('#your-date-picker'))
datePicker.bootstrap()
```

## Usage

```js
console.log(datePicker.value) // for example 2023-08-22. Per default current day.
datePicker.value = '2024-01-01' // programmatically set the value
```
